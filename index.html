Project 01

Credit Risk Case Study – Categorical Data Analysis

Simulated a bank-level credit scoring system using 1,000 observations and 20 predictors. Performed EDA with correlation matrices and conditional proportion plots. Ran stepwise AIC model selection (forward, backward, both — all converged), then validated with the Hosmer-Lemeshow goodness-of-fit test and ROC curve analysis.

90% Test Accuracy ROC AUC 0.844

Project 02

Machine Learning – Text Classification

Implemented and benchmarked multiple classifiers to automatically label news articles into five categories: business, entertainment, politics, sport, and technology. Trained and compared Naïve Bayes, Gaussian class conditional, and k-NN algorithms, analyzing precision and recall performance across all labels.

Up to 75% Accuracy 5 Categories 3 Algorithms
Python Naïve Bayes Gaussian k-NN NLP Multi-class Classification Text Feature Extraction
Project 03

UFC Causal Inference – Effect of KO/TKO Losses

Web-scraped 5,000 recent UFC fights, cleaned and organized data by date and weight class. Applied propensity score matching and regression to determine that age and prior KO/TKO losses are the most significant causal factors influencing future stoppages. Validated findings with a predictive betting model.

70% Betting Accuracy 5,000 Fights
